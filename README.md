1. Project Title

Corporate Website named: Top to Toe | Natural Beauty Salon

2. Description

A responsive website built with HTML5, CSS3, Tailwind CSS and JavaScript.
It showcases services, for the Business.

3. Features

• Responsive design (mobile, tablet & desktop friendly)
• Accessible with proper semantic HTML and ARIA labels
• Sticky navigation bar and hamburger menu
• Custom branding and styling

6. Accessibility & Best Practices

This site follows accessibility best practices:

- Semantic HTML tags
- Proper alt attributes for images
- Language attribute set to `en-GB`
